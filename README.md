# Trang web quản lý Quán Cafe
## Case study module 3
## Các kiến thức được áp dụng:
1. HTML/CSS
2. JavaScript
3. Bootstrap
4. Java Servlet
5. JSP/JSTL
6. MySQL Database
## Mô tả chức năng:
- Chức năng quản lý Nhân viên, bao gồm:
  - Thêm, sửa nhân viên
  - Phân quyền nhân viên với 4 nhiệm vụ khác nhau: quản lý, thu ngân, pha chế, phục vụ
  - Xóa đi các nhân viên đã thôi việc
- Chức năng quản lý thức uống, bao gồm:
  - Thêm, sửa thức uống
  - Ẩn hoặc hiện thức uống theo nhu cầu chủ quán
- Chức năng quản Tài khoản đăng nhập hệ thống
  - Phân quyền tài khoản theo 2 cấp độ: admin (quản lý), user (thu ngân)
  - Tài khoản admin có thể thực hiện đầy đủ các chức năng của hệ thống như: cấp tài khoản mới cho nhân viên đang làm việc, thêm/ sửa/ xóa tài khoản
  - Tài khoản user dành cho nhân viên thu ngân, chỉ có chức năng quản lý order
- Chức năng quản lý order:
  - Tạo order mới. Không thể sửa hay xóa các order trước đó
  - Thông tin chi tiết order thể hiện đầy đủ giá tiền, tổng tiền, ngày tạo, người tạo
## Các chức năng còn thiếu cần bổ sung:
- Chức năng tìm kiếm cho các mục quản lý
- Chức năng upload ảnh để thể hiện đầy đủ hơn mô tả thức uống, thông tin nhân viên cũng như tạo sự cá nhân hóa cho các user
- Chức năng áp dụng mã giảm giá khi order thức uống
- Chức năng thống kê
- Chức năng in ấn
